###### 개발 배경

운전자의 운전중 모습을 관찰하고 피드백하여 올바른 운전 문화를 조성하고자 하며, 가까운 운전자들 간에 위험 운전 행위를 공유하여 사고를 사전에 방지하고자 합니다.
